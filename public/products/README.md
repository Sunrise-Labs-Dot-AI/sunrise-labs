# Product screenshots

Drop app screenshots here. Paths referenced from `src/data/products.ts`.

## Expected files

- `engrams.png` — browser-frame screenshot of getengrams.com. Roughly 16:10 aspect; any resolution ≥ 1440×900 is fine (rendered with `object-cover`).
- `sitter.png` — phone-frame screenshot of a Sitter SMS conversation. Roughly 9:19.5 aspect (iOS screenshot works as-is).

If a file is missing the frame still renders — the empty slot reads as a placeholder.
